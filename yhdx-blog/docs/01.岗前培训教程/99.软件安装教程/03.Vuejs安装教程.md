### 一. 环境准备

### 1. 脚手架运行环境

1）Node.js运行环境；

2）npm（node package manage）依赖包

**脚手架的好处：**

1）脚手架可以大大的提高开发效率；

2）可使用最主流的ECMAScript语法；

3）通过webpack实现编译查看效果（非浏览器编译）；

4）自动更新，可实时查看最新效果等；

**依赖安装**

```shell
// 安装脚手架
npm install -g @vue/cli
// 查看版本
vue --version
```

### 2. Vue-devtools 插件

#### 安装

1. 科学上网后进入 `https://chrome.google.com/webstore/category/extensions`
2. 搜索 Vue
3. 点击 `添加至Chrome`
